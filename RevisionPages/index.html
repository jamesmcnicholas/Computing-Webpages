About The Site

This set of webpages was created to help summarise content on the A2 Computer Science specification for OCR

The content is split into topics which can be viewed via the navbar at the top. The sections are as follows:

  • Processing -> GPUs, Pipelining, Parallel Processing
  • Systems -> Database Normalisations and Transactions
  • Programming -> OOP Fundamentals, Inheritance, Types of Languages, Recursion
  • Development -> Stages of Compilation
  • Data -> Floating Pint Arithmetic, Advanced Boolean Algebra, Boolean Applications
  • Computation -> Trees and Graphs, Advanced Sorts, Path Finding, Algorithms and Efficiency, Computational Methods
  • Connectivity -> Compression, Network Security, Encryption and Hashing, Web Technologies

Each page is split into subtopics, each of which contains the basic knowledge about that subject. These do not go into the full detail necessarily required for the exam, but should give you a basic idea of most of the concepts. From this you should be able to remind yourself and then progress or research further into areas you are unsure about.
